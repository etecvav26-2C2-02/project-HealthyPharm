# project-HealthyPharm

## Tema: 
Saúde, farmacologia, e a disseminação de remédios desconhecidos, para o conhecimento popular.

## Proposta:
* Busca criar uma comunidade que expalha medicamentos desconhecidos ou impopulares;
* Os usuários encontrará uma aba onde poderá cadastrar um novo medicamentos que ele considera relevante;
* O site tenta apontar opções mais acessivéis de soluções pra doenças diversas;
* Ele irá oferecer informações de pessoas sobre possivéis medicamentos para utilizar.

## Público-alvo:
* Principalmente adultos entre 20 a 25 anos que recém são pais e desconhecem opções acessivéis;
* Com o interesse de apoiar a comunidade de jovens pais;
* Normalmente pessoas dessa idade com filhos ainda estão estruturando uma base sólida monetária, e os filhos são propícios a serem afetados por infecções entre outros, e medicamentos especifícos são bem caros.
