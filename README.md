## 4️⃣ Aplicar o Conceito Mobile First
O projeto seguirá o conceito Mobile First, pois grande parte dos usuários acessa informações de saúde e medicamentos pelo celular. Dessa forma, o site será desenvolvido inicialmente para dispositivos móveis, garantindo praticidade e rapidez no acesso às informações.

Como o site será adaptado para celular:
Layout simples e organizado.
Navegação intuitiva e fácil de usar.
Botões grandes para facilitar o toque na tela.

Carregamento rápido das páginas.
Informações importantes destacadas para fácil visualização.
Cuidados que serão tomados:
Menu: menu compacto (hambúrguer) para ocupar menos espaço.

Imagens: imagens leves e bem dimensionadas para não prejudicar o carregamento.
Textos: uso de fontes legíveis e tamanho adequado para leitura em telas pequenas.
Organização da tela: conteúdo disposto de forma vertical para facilitar a navegação e a localização das informações.
